Routes Table

HTTP Method Description Endpoint
GET Retrieves all the users in the database /users
GET Retrieves a single user by ID /user/:id
POST Registers a new user /register
POST Logs in a user /login
PUT Updates user details /user/:id
DELETE Deletes a user by ID /user/:id
GET Retrieves all projects for a specific user /user/:userID/projects
POST Creates a new project for a user /user/:userID/project
PUT Updates a project for a user /user/:userID/project/:projectID
DELETE Deletes a project for a user /user/:userID/project/:projectID
GET Retrieves all notes for a specific user /user/:userID/notes
POST Creates a new note for a user /user/:userID/note
PUT Updates a note for a user /user/:userID/note/:noteID
DELETE Deletes a note for a user /user/:userID/note/:noteID
GET Retrieves all courses for a specific user /user/:userID/courses
POST Creates a new course for a user /user/:userID/course
PUT Updates a course for a user /user/:userID/course/:courseID
DELETE Deletes a course for a user /user/:userID/course/:courseID
